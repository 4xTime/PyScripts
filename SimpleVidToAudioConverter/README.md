#### install requirements.txt

```http
  pip3 install -r requirements.txt
```

| Flag | Description                       |
| :-------- || :-------------------------------- |
| `--URL`      || URL of youtube video |


| Flag | Description                       |
| :-------- || :-------------------------------- |
| `--ConvertFile`|| Path to your file |

| Flag | Description                       |
| :-------- || :-------------------------------- |
| `--Format`||  Your desire file format|

| Flag | Description                       |
| :-------- || :-------------------------------- |
| `--DEL`|| Delete video file after conversion|
